## Install

```bash
npx skills add https://github.com/balajmarius/benln-cold-email-outreach
```

## What It Does

The skill writes cold emails with these constraints:

- 2-3 sentences
- read like a text
- have a clear ask
- no signature
- thoughtfully personalized
- punchy subject line
- avoid vague asks like "pick your brain"
- do not sound like AI wrote them
